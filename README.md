# DT208G - Typescript

Detta är en webbapplikation för att hantera kurser genom att lägga till och radera.
Datan sparas med hjälp av localStorage.

## Funktionalitet
- Lägga till kurs
- Radera kurs
- Visa tabell
- Validering av kurskod, går inte att lägga till samma kurskod två gånger
- Spara data med localStorage

### Funktioner
- Typescript
- Vite
- HTML
- CSS
- DOM
